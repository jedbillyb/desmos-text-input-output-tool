# Desmos Text I/O

Import/export Desmos graphs as JSON text, for Chrome and Firefox.

## Features
- Export any Desmos graph as JSON
- Import JSON back into Desmos
- Works on https://www.desmos.com

## Installation

### Firefox
1. Download the latest release
2. Go to `about:debugging` → This Firefox → Load Temporary Add-on
3. Select `manifest.json`

### Chrome
Install from the Chrome Web Store *(link to come soon)*

## Usage
1. Open a Desmos graph
2. Click the extension icon
3. Hit **Export** to copy the graph state as JSON
4. Edit if needed, then hit **Import** to load it back
